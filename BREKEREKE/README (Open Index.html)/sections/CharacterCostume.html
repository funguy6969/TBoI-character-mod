Character Template ReadMe

Character Costume(s) and Character Skin

In this section we will be working on the costume(s) for your character(s) and the Character(s) skin. We will start with the skin.

The file we will be working on can be found in .\resources\gfx\characters\costumes

I highly recommended to use my template and edit it instead of trying to make your own!

If you do decide to make your own, the file needs to be 512x512

When you are done save the file as character_charactername.png

Now do this again for your tainted character (or just copy it if you want them to be the same) and name it character_charactername_b.png

Character Costume

Next we will create an overlay costume for the character (Like maggie's hair, or cain's eyepatch)

If you do plan on making a costume, you will still need to delete some files.

  • costumes2.xml
  • character_alpha_cat_ears.anm2
  • character_cat_ears_template.png

Regardless if you do a costume or not be sure to delete the character_cat_ears_template_reference.psd file when your done using it (if you do).

Included in this template is a simple cat ears costume. along with its PSD reference I used to create it.

This template allows for applying costumes to the tainted and regular character separately.

You must put all the different files I made in the same place as I put them.

anm2s in \character and pngs in \character\costumes.

Creating a costume takes 4 steps.

  1. Make the costume frames in .\resources\gfx\characters\costumes
    2. - you need 8 frames total, 1 for each each facing direction + 1 for each direction while firing
  2. Setup the anm2 in .\resources\gfx\characters
    3. - you can use the animation editor in the tools folder of the Isaac install (This is covered in Character menu)
  3. Tell the game your costume exists by adding it to costumes2.xml found in .\content
  4. Apply the costume in the stats.lua file
