Character Template ReadMe

Why is my image appearing as a messed up / black square?

This is because you need to save your image as a 32-bit (8bpc) png.

Is there any way to make my code easier to read?

Yes! Use a code editor to edit your code. A good free one is Notepad++

How do I get to where I have installed Isaac?

If you don't know how to get to your Isaac install, do this.

  1. Open Steam
  2. Right click the game
  3. Go down to Manage
  4. Click Browse Local Files

Can I use the default error checker?

Yes. You can disable my custom error checker in the main.lua file, at the very top just set the custom error checker variable to false!

What if I don't want to make a tainted Character?

Simple, in stats.lua set the enabled variable of the tainted character to false. then in players.xml just remove the second tainted character.

Imports.lua?

This is a file for adding in content from other files. In other words its for those who know what to do with it. If you don't need it just leave it be.
