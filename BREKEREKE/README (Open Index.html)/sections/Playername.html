"Player Name" Image

In this section the goal will be to create the image that shows up for you enter a boss arena, and similiarly related screens.

To create the base image you can use the Isaac Text Tool

The file that we will be creating can be found in .\resources\gfx\ui\boss\

When on the tool you will want to switch the text type to "VS Screen", type in your characters name, then right click it and save image as.

Next we will need to edit the image for our purpose. Open the image in your editor of choice and do these steps.

  1. Crop the image to be exactly 192 x 64, this is the image size the game uses for the PlayerName Image
  2. If your editor has the tools, true align the text to be centered on both the X and Y
  3. Save the file as playername_charactername.png (lowercase the name)
  4. you also need to make one for your tainted character, name it playername_charactername_b.png
